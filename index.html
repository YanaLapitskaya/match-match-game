RULES

Memory is a game, where the goal is to find pairs to pictures.

When the game begins, all pictures are hidden. It is allow to open only 2 pictures in the same time

HOW TO PLAY

  1. Select 2 cards to try to match the pictures
  2. If pictures are the same, they disappear from the desk
  3. If they don't match, they become hidden
  4. The player that finds all pairs wins!
  5. Have fun!
